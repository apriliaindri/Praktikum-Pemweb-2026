# Tugas 2

## Deskripsi Tugas

Pada tugas ini, kalian diminta untuk membuat atau mengembangkan sebuah website sederhana dengan **menambahkan fitur interaktif menggunakan jQuery**. Ketentuan tugas:
1. Kalian boleh mengembangkan website dari tugas sebelumnya atau membuat website baru dengan tema bebas.
2. Website harus menggunakan HTML, CSS, dan jQuery.

## Rubrik Penilaian

Penilaian tugas akan didasarkan pada beberapa aspek berikut:
1. Kerapihan Code (kode tersusun dengan rapi, mudah dibaca, dan memiliki struktur yang jelas).
2. Kelengkapan Ketentuan (website telah memenuhi seluruh ketentuan yang diminta).

## Pengumpulan

Tugas dikumpulkan dalam 2 format:
1. Laporan dalam bentuk Portable Document File (\*.pdf) dengan ketentuan nama file **PPW_Kelas_02_NIM_NamaLengkap.pdf** 
2. Source code dalam bentuk file ZIP yang berisi file HTML, CSS, dan JavaScript (jQuery) dengan nama berkas **PPW_Kelas_02_NIM_NamaLengkap.zip**
