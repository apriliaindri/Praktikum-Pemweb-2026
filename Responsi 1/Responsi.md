# Responsi 1

## Deskripsi Tugas

Mahasiswa diminta untuk membuat sebuah website interaktif dengan tema bebas dengan mengimplementasikan seluruh materi yang telah dipelajari selama praktikum. Website harus mampu menampilkan interaksi dinamis, manipulasi DOM, serta komunikasi data secara asynchronous menggunakan AJAX.

## Ketentuan Umum
Website yang dibuat wajib mengimplementasikan:
1. HTML
2. CSS (menggunakan file CSS terpisah)
3. JavaScript dan DOM, manipulasi elemen HTML menggunakan DOM, seperti:
   - Menambah atau menghapus elemen
   - Validasi form
   - Event handling (click, submit, dan lainnya)
4. jQuery, contoh implementasi:
   - Seleksi elemen
   - Event handling
   - Animasi sederhana (fade, slide, dan lainnya)
5. AJAX, contoh implementasi:
   - Mengambil data dari API
   - Submit form tanpa reload
   - Menampilkan data dinamis dari server atau file JSON

## Rubrik Penilaian

Penilaian tugas akan didasarkan pada beberapa aspek berikut:
1. Kerapihan Code (kode tersusun dengan rapi, mudah dibaca, dan memiliki struktur yang jelas).
2. Kelengkapan Ketentuan (website telah memenuhi seluruh ketentuan yang diminta).
