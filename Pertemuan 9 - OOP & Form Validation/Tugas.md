# Tugas 3

## Deskripsi Tugas

Pada tugas ini, kalian diminta untuk membuat atau mengembangkan sebuah website sederhana menggunakan **PHP Native, Database MySQL, OOP, dan Form Validation**. Ketentuan tugas:

1. Kalian boleh mengembangkan website dari tugas sebelumnya atau membuat website baru dengan tema bebas.
2. Website harus menggunakan **PHP Native** (tidak diperbolehkan menggunakan framework PHP).
3. Website harus terhubung dengan **Database MySQL**.
4. Implementasikan fitur **CRUD** pada minimal satu data utama.
5. Seluruh proses pengelolaan data harus menggunakan konsep **OOP**.
6. Terapkan **Form Validation** pada halaman input data atau halaman login/registrasi untuk memastikan data yang dimasukkan sesuai dengan ketentuan.
7. Boleh menggunakan Bootstrap atau CSS untuk mempercantik tampilan website.

## Rubrik Penilaian

Penilaian tugas akan didasarkan pada beberapa aspek berikut:

1. Kerapihan Code (kode tersusun dengan rapi, mudah dibaca, dan memiliki struktur yang jelas).
2. Kelengkapan Ketentuan (website telah memenuhi seluruh ketentuan yang diminta).
